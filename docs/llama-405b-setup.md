# Llama 405B setup

Llama 3.1 405B remains fully supported, but it is now the primary reasoning model rather than the only model. Use it for business strategy, research, BRDs/PRDs, product planning, architecture review, and final implementation review.

Recommended production profile: 8x H100 80GB, 8x H200, 8x B200, or equivalent multi-GPU RunPod profile with at least 640GB total VRAM and 2TB model cache.
