# Purpose

This phase defines the AI Coding Workspace.

The Coding Workspace is not an editor.

It is a complete AI Software Engineering environment capable of planning,
building, reviewing, testing, deploying and maintaining software.

The Coding Workspace integrates with Projects, Conversations, Workflows,
Knowledge, Runtime Management and AI.
